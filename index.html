About This Site

This static website is running on an Amazon EC2 instance. The web server (Apache/Nginx) serves this page directly from the /var/www/html directory.

How It Works

  • Launch an EC2 instance (Amazon Linux or Ubuntu)
  • Install Apache or Nginx to serve web content
  • Configure Security Groups to allow HTTP traffic (port 80)
  • Upload website files to /var/www/html
  • Access the site via the EC2 Public IP or domain name

Get Started

Hosting on EC2 is perfect for learning Linux, configuring servers, and running small-scale web projects with full control over your environment.
